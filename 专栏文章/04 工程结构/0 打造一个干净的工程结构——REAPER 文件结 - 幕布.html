0 打造一个干净的工程结构——REAPER 文件结构基础设置
  • 拷贝文件到工程目录下
    • Preference: Media/Copy imported media to project directory
  • 设置工程的媒体文件夹
    • Project Settings: Media/Path save..
    • 记得把这个设置作为所有工程的默认设置
  • 在保存工程的时候复制所有相关文件
    • Save as 时勾选 Copy..
  • 清理工程中不用的媒体文件
    • File: Clean current project directory..
  • （顺便）查看/搜索工程中正在使用的媒体文件和使用状况
    • Project Bay 窗口
  • 把所有波形文件都放到另外的文件夹中
    • Preferences: Paths/Store all peak caches...
  • 工程备份设置
    • 每隔多久保存一次
    • 把文件关联设定为相对路径
    • 是否开启多版本？是否保存时带上时间戳？
    • 储存在专门的备份文件夹内
  • （顺便）默认的导出路径、录音路径
    • 默认导出路径：Preferences: Paths/render path
    • 默认录音路径：Project Settings/Secondary recording paths
    • 默认的录音名称格式
